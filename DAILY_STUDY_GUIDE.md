# ENEM 2026 Daily Study Guide

## Quick Start

Your daily study journals are now populated with comprehensive study plans! Here's how to use them effectively.

Navigate to: `docs/en/journal/[year]/[month]/[date].md`

For example:
- November 5, 2025: `docs/en/journal/2025/11-november/2025-11-05.md`

## Daily Structure (8 Hours Total)

Each day is organized into 6 study blocks:

### 1. Portuguese (1.5 hours)
- **Focus**: Grammar, text interpretation, literature
- **Activities**: Reading exercises, ENEM questions, writing practice
- **Check**: Goals are listed as checkboxes - mark them off as you complete!

### 2. Mathematics (2.0 hours)
- **Focus**: Algebra, functions, geometry, problem-solving
- **Activities**: Problem sets, graphing, applications
- **Tip**: This gets the most time - math builds on itself!

### 3. Natural Sciences (1.5 hours)
- **Focus**: Biology, Chemistry, Physics (rotating)
- **Activities**: Reading, diagrams, experiments, questions
- **Note**: Topics rotate to cover all three sciences

### 4. Humanities (1.5 hours)
- **Focus**: History, Geography, Philosophy, Sociology (rotating)
- **Activities**: Timelines, maps, concept analysis
- **Connect**: Think about how topics relate to modern Brazil

### 5. Essay Writing (1.0 hour)
- **Focus**: ENEM essay structure and techniques
- **Activities**: Paragraph practice, full essays, analysis
- **Key**: Practice the 5 competencies regularly!

### 6. Computational Neuroscience (0.5 hours)
- **Focus**: Building neuroscience foundations
- **Activities**: Video lectures, reading, coding
- **Goal**: Connect to your UFABC dream!

## How to Use Each Day

### Morning Routine

1. **Read today's journal entry**
   - Review all subject topics
   - Note the learning objectives
   - Gather needed materials

2. **Set up your workspace**
   - Have notebooks ready
   - Open resource links in browser
   - Set timer for each block

3. **Start with your strongest subject**
   - Build confidence early
   - Use momentum for harder subjects later

### During Study Sessions

1. **Work through each subject block**
   - Check off goals as you complete them
   - Write notes in the journal
   - Track actual time spent

2. **Use the provided resources**
   - Links are at the bottom of each day
   - Choose resources that work for your learning style
   - Bookmark favorites

3. **Take breaks between subjects**
   - 5-10 minute breaks
   - Stay hydrated
   - Stretch and move

### Evening Review

1. **Update Daily Progress section**
   - List completed tasks
   - Calculate total study hours
   - Note achievements and struggles

2. **Fill in Daily Reflection**
   - What did you learn?
   - How did you feel?
   - What challenged you?

3. **Review Tomorrow's Planning**
   - See what's coming next
   - Prepare needed materials
   - Set goals for tomorrow

5. **Practice exams**: Use past ENEM questions regularly
